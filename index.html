Sobre Mim

Minha Jornada

Sou formado em Informática e desde então, venho aprofundando meus conhecimentos nas áreas de Análise de Dados, Python e HelpDesk.

Meu estilo de aprendizado é prático e investigativo — gosto de entender os processos por trás das ferramentas e aplicá-los em projetos que geram impacto real. Já desenvolvi soluções como chatbots para WhatsApp, automações em Python, geradores de relatórios e até modelos de previsão com IA.

Nos últimos meses, venho me dedicando especialmente à análise de dados, criando dashboards com Power BI e explorando dados de vendas no varejo para gerar insights estratégicos.

Sou curioso, determinado e gosto de trabalhar com desafios. Este portfólio é um reflexo da minha trajetória, onde cada projeto representa uma etapa da minha evolução profissional e pessoal.

Minha abordagem combina habilidades técnicas sólidas com uma compreensão profunda dos negócios, permitindo-me criar soluções que realmente fazem a diferença.

Análise de Varejo Comercial

Neste projeto, analisei dados de vendas de uma rede de varejo, identificando tendências e padrões que ajudaram a entender melhor quais decisões a tomar, resolvendo 10 perguntas de negócios.

Python CSV Pandas NumPy Seaborn Matplotlib

Análise de Indicadores Econômicos de Boston (2013–2019)

Neste projeto, analisei os principais indicadores econômicos de Boston entre 2013 e 2019, como taxa de desemprego, PIB e inflação, passagens aéreas e ocupação dos hotéis. Através de gráficos, é possível visualizar as tendências e correlações entre esses indicadores.

Python Excel Pandas NumPy Seaborn Matplotlib

Chatbot para WhatsApp

Este projeto implementa um chatbot no WhatsApp utilizando a biblioteca venom-bot . O chatbot foi desenvolvido para interagir com alunos do ISPLB, fornecendo informações úteis sobre cursos, propinas, contas bancárias, localização e outras funcionalidades. Ele oferece uma experiência interativa e pode ser configurada para atender à demanda de diferentes usuários.

JavaScript VenonBot SQL NodeJS GitHub
